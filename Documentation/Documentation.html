Getting Started #back to top

When you are ready to use this template, you must first upload the template files to a local or remote server. In this regard, unzip the template package you have downloaded from Themeforest. In extracted folder you will find following folders:

  • 1. Template

    Contains all the original template files comprises of HTML, CSS, JS etc. This is the folder which you'll upload on your server after necessary modifications.

  • 2. Documentation

    Inside this folder you will find this same documentation updated with the last changes.

Template Structure #back to top

This template was built with the help of Twitter Bootstrap. Open the TEMPLATE folder and your will find all HTML, CSS, JS and PHP files that you'll need to customize to add your content. This template consist of 3 handcrafted HTML pages. All of these pages can be found at the root of the template folder.

How to Publish Template #back to top

Upload via FTP Server

The best way to upload your web site is via FTP. This can be done in a number of different ways using many different FTP clients, we suggest FileZilla.

  1. With FTP connect to the website server.
  2. With FTP upload files on the website server.
  3. Open internet browser.
  4. Enter your website url

Using an FTP program, upload only the unzipped template folder/files on your website server.
